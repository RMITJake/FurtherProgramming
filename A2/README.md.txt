# s3905550 - Further Programming A2
## Requirements
Java 21.0.3 SDK https://www.oracle.com/au/java/technologies/downloads/#java21
JavaFX 21.0.3 https://gluonhq.com/products/javafx/
SQLite-JDBC 3.45.3.0 https://github.com/xerial/sqlite-jdbc/releases

## Running the Program
### Run the program in VSCode
1) Click File > Open Folder.
2) Browse to the location of the A2 folder and open the A2 folder.
3) Open the src/App.java class and click the play button.

### Run the program in Eclipse
1) Click File > Open Projects from File System.
2) To the right of the Import Source field select "Directory"
3) Browse to the location of the A2 folder and import the A2 folder.
4) Select the A2 Project and click the Play button.
